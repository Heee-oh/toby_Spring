정통 빈 등록의 경우 XML을 이용하여 등록함

경로 : src/main/resoucres/applicationContext.xml

```xml
<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="
        http://www.springframework.org/schema/beans
        https://www.springframework.org/schema/beans/spring-beans.xsd">
    <bean id="bean_등록_id" name="별칭 (여러개로 호출가능)" class="toby.toby_spring.ch5.User"> <!-- 실제 클래스 경로 -->
        <property name="repository" ref="userDaoJdbc"/>
        
    </bean>
    
    <bean id="userDaoJdbc" class="toby.toby_spring.ch5.UserDaoJdbc">
        <constructor-arg ref="datasource"/> <!-- 생성자 주입-->
        
    </bean>
</beans>
```
기본적으로 xml 등록은 Setter 주입이다.

<property name="" />
name 속성은 setter 메서드의 이름을 지정한다. 
name="ex" -> setEx() 메서드 호출

ref : XML에 값으로 넣은 bean id를 찾아서 주입한다.

value : 일반 값(데이터) 주입, String, int 등


하지만 constructor-arg 로 생성자 주입이 가능하다
매개변수가 여러개인 경우
index="N" 속성을 넣어 파라미터 순서를 맞춘다. (0부터 시작)


그외 더 많지만 스프링부트가 현재는 자동화해주고, xml로 일일히 등록하는것도 현재로써는 번거롭다.
Component, Configuration, @Bean 등으로 편리하고 빠르게 등록가능
