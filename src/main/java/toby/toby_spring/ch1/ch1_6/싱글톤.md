싱글톤 레지스트리 : 싱글톤을 저장하고 관리하는 애플리케이션 컨텍스트
- 평범한 자바 클래스를 싱글톤으로 활용하게 해줌
- 

스프링의 싱글톤이 아닌 싱글톤 패턴에 대해서 

싱글톤 패턴 구현방법
- 생성자 private
- 생성된 싱글톤 오브젝트를 저장할 수 있는 자신과 같은 타입의 static 필드 정의
- 스태틱 팩토리 메서드인 getInstance() 생성, 1번만 오브젝트 생성, 이후 반환

문제점
- private 생성자로 인한 상속 불가
- 테스트 어려움
  - Mock 오브젝트 등으로 대체 어려움
- 서버 환경에서는 싱글톤이 하나만 만들어지는 것을 보장하지 못함
- 싱글톤의 사용은 전역 상태를 만들 수 있음
  - 전역 상태 : 아무 객체나 자유롭게 접근, 수정, 공유할 수 있는 상태
  - 객체 지향에서 권장하지 않음
